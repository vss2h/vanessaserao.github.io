Overview

The noise detector is a project created for Middle Tennessee States University. The purpose of the project was intended to monitor sound in the quiet zones of the universityâ€™s library.
As the staff at the library was getting many complaints and concerns by students about the quiet zones being noisy at times, a tool was needed to come in aid of those students to circumvent these complaints.

Tools & Usability

Through the creation of this project, many tools such as Raspberry Pi 3 (image 1), noise measurement monitor (image 2&3), LAMP server, and Adobe Dreamweaver were used. The project was composed of a web application, sound monitor, and a database.

The Raspberry Pi 3 was used to create the main foundation of the project. It hosted the web application through its LAMP server, and it also stored the data retrieved from the sound monitor into its database. The web application was created using Adobe Dreamweaver.

Functionality

The project is placed in different quiet areas of the library to detect any loud and disruptive noises. The noise measurement monitor receives different noise levels and compares it to the thresholds. The noise is not processed until a certain level of it is reached. Once this level of threshold is reached, the information providing the time, date, and noise level is stored in the database within the web application as well as sending an email with the information to the library staff.
