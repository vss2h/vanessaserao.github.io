Led Cube

Overview

The LED Cube is a project created for the Middle Tennessee State University's MakerSpace.
It was created as a display model to trigger student's creativity and inspire research.

The purpose of this project was to perfom beta testing on neopixels (individually-addressable RGB color pixels) in order for the library technology to see how they work, what they could be used for, and if they could be bought for student's use.

The creation of this project involved different skills varying from electronics to new programming language to building hardware that were learned through research and reading.

Software & Tools

  • Neopixels
  • Arduino Mega
  • Electronic circuit
  • C programming language

Process

Not being familiar with electronics, building hardware, or any of the tools used for the accomplishment of this project, the project was a interesting challenge. A lot of research, tries, and tests were put into it.
For learning purpose, a small prototype was built, which made understanding the process better.
