Trivia Game

Overview

The trivia game was team project created at the 2018 Middle Tennessee State University Hackathon (HackMT).
The idea was to create a multi-player trivia game with questions about MTSU's campus. The project was completed in 35 hours and included both hardware and software.

Tools and Usability

The tools and languages used to accomplish the project are:

  • Python
  • Raspberry Pi 3
  • Pygame
  • Game buttons

The game was built in the Raspberry Pi 3, written in Python using Pygame, a module designed for writing video games.

To allow the multi-player feature, we built two game buttons using old boxes, breadboards and red buttons and connected them to the Raspberry Pi.

Challenges and Accomplishment

As exciting and fun the project was, the group encountered many difficulties along the way.

From learning and adjusting to different tools on the fly, how they function, and managing to create a fully functional project in a few hours; we had to work as a team and not only split tasks but also know enough about everything else to help each other when encountering a issue.

By the end of the Hackathon, we were able to show the public our functioning application and have them test the game by playing against each other.
